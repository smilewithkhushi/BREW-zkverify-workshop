## Contracts

### `IVerifyProofAggregation.sol`
Interface for ZKVerify's on-chain aggregation verifier proxy on Sepolia.
- Proxy address: `0xEA0A0f1EfB1088F4ff0Def03741Cb2C64F89361E`

### `MagicSquareVerifier.sol`
Main verifier contract. Computes the UltraHonk leaf digest and calls ZKVerify's proxy to verify the aggregation proof on-chain.

## How to Verify

1. Open [Remix](https://remix.ethereum.org) and load both `.sol` files
2. Connect SubWallet on **Sepolia** via Injected Provider
3. Use "At Address" with `0x8e216b21ac0CE2c9D3114547bd9d4b098BDA5457` to load the deployed contract
4. Call `checkHash` with the parameters above
5. Returns `true` = proof verified on-chain
